# Visualisation des données 
Ce projet est dédié à la visualisation graphique des données au moyen des deux librairies matplotlib et plotly.

## Description des fichiers :

#### test_trace_2D_statiques.py 
- Tracé 2D statique simple 
- Tracé 2D statique somme cumulée
- Tracé 2D avec style et échelle personnalisé
- Tracé 2D avec limites d'axes spécifiques
- Tracé 2D avec taille et labels personnalisée
- Tracé 2D de deux jeux de donnée
- Tracé 2D simple avec deux jeux de données légendés
- Tracé 2D avec deux jeux de données et problèmes d\'échelle
- Tracé 2D avec deux jeux de données et résolution d\'échelle
- Tracé 2D avec 2 sous-tracés séparés
- Tracé combiné : Lignes et Barres
- Scatter plot type plt.plot
- Scatter plot type plt.scatter
- Scatter plot avec échelle de couleur
- Histogramme 2D statique avec deux séries de données
- Histogramme 2D statique empilant deux jeux de données
- Boîte à moustache pour deux jeux de données
- Tracé d\'une fonction exponentielle avec intégrale

#### test_trace_3D_statiques.py 
- Volatilité implicite en 3D
- Tracé d\'un nuage de points 3D avec les volatilités induites

#### test_trace_2D_interactifs.py 
- Tracé en ligne séries temporelles
- Histogramme des rendements quotidiens
- Box plot des rendements quotidiens
- Graphique en chandelier (candlestick)
